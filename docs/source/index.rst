**AMPcombi** and its submodules provide a command-line interface to parse the results of antimicrobial peptide (AMP) prediction tools into a single table, aligns the AMP hits against reference AMP databases for functional and structural classifications, filters the AMP hits according to their physiochemical properties, clusters the filtered hits and predicts signaling peptides if present.

The input data for AMPcombi can be generated by running the individual AMP prediction tools described in (:ref:`usage`). AMPcombi can be used as a standalone tool (see :ref:`usage`) or in pipelines like `nf-core/funcscan <https://github.com/nf-core/funcscan>`_.

AMPcombi source code is hosted on `Github <https://github.com/paleobiotechnology/AMPcombi>`_. The tool has been published in the **Journal of the American Chemical Society**: `10.1021/jacs.5c14335 <https://doi.org/10.1021/jacs.5c14335>`_. If you use AMPcombi for your research please don't forget to **cite** us. 💙
